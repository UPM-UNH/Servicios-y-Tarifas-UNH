# Tarifas UNH – TUPA / TUSNE

Widget web para la consulta interactiva de tarifas TUPA y TUSNE
de la Universidad Nacional de Huancavelica.

La información se obtiene desde Google Sheets en formato CSV
y se muestra con filtros, búsqueda y exportación a PDF.
